Development Phases

LEVEL 0

CURRENT VERSION

Fully teleoperated. Control via keyboard and leader robot arm. No autonomy, no vision - just a solid working base.

LEVEL 1

BASIC AUTONOMY

LiDAR mapping, SLAM navigation, inverse kinematics, and basic presets like "grab" or "place."

LEVEL 2

INTELLIGENT ROUTINES

Computer vision, object recognition, voice control, and memory for complex tasks.

FAQs

Is Raven actually open source?

Absolutely! All hardware designs, software code, and build instructions will be available on GitHub. The goal is to create a community-driven platform for home robotics.

How much does it cost to build one?

The current BOM is around â‚¹20,000-40,000 (approximately $250-500 USD). We're working on a dev kit that we'll keep you updated about.

Can it actually find my glasses?

Not yet! The current Level 0 version is teleoperated only. Computer vision and object recognition are planned for Level 2.

When will the dev kit be available?

We're working on getting to Level 2 before we work on the dev kit. Join our Discord to get updates on availability!

Can I contribute to the project?

Yes! Whether you're into hardware, software, or just have ideas, there's room for contribution. Follow the GitHub repository and join our Discord community.

What programming experience do I need?

Basic Python knowledge is helpful for customization. The system uses ROS 2 and the LeRobot SDK, but we're working on making it accessible to non-programmers.
